# Fisheye

Real-time video lens distortion effects using TypeGPU.

## Setup

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in a WebGPU-compatible browser 

## Controls

- **Vignette**: Edge darkening
- **Lens Distortion**: Fisheye effect (-5 to 5)
- **Scale**: Zoom level
